Référence des inputs

Input Type Défaut Description
selectionMode 'date' | 'month' | 'year' 'date' Granularité de la sélection
enableTimePicker boolean false Affiche les champs Start / End time
presets DateRangePreset[] 9 presets Liste de raccourcis de période
showPresets boolean true Affiche / masque le panneau presets
showModeSelector boolean true Affiche / masque le sélecteur Jours / Mois / Années
layout 'auto' | 'horizontal' | 'vertical' 'auto' Disposition du panneau (auto = responsive)
placeholder string 'Select a date range' Texte affiché dans le champ vide
min Date | null null Date minimale sélectionnable
max Date | null null Date maximale sélectionnable
disabled boolean false Désactive le composant (CVA compatible)
required boolean false Marque le champ comme obligatoire

Outputs

Output Type Description
rangeChanged EventEmitter<DateRangeResult> Émis à chaque clic sur Apply
opened EventEmitter<void> Émis à l'ouverture du panneau
closed EventEmitter<void> Émis à la fermeture du panneau
